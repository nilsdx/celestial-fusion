# Dual Bird

Dual Bird are rare Mechguns that are [Master Raven](https://wiki.pioneer2.net/w/Master_Raven) and [Last Swan](https://wiki.pioneer2.net/w/Last_Swan) combined together. It is not a Destiny exclusive weapon.

When [Jointparts](/items/jointparts) are equipped, they become strong long range single target weapons on Hunter characters.

## Set Effect

| Item | Effect |
| ----------- | ----------- |
| [Jointparts](/items/jointparts) | Range +30.0, Hits: 4-4-4, ATA +20 |
