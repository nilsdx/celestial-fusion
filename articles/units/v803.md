# V803

V803 is a rare unit, and is a direct upgrade to V802, offering the same casting speed increase, along with powerful Freezing chances increases.

While not a mandatory item, if you have enough Millenium Photon Spheres and haven't made the other top tier endgame items, definitely consider making this unit.

## Technique Boosts

| Technique | Boost |
| ----------- | ----------- |
| Gibarta | Freeze success rate: 90% |
| Rabarta | Freeze success rate: 50% |
| Gibarta (HU/RA) | Freeze success rate: 66% |
| Rabarta (HU/RA) | Freeze success rate: 33% |
