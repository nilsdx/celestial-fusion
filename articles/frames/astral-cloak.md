# Astral Cloak

Astral Cloak is a rare frame that boosts all Si techniques's power by 100% and all Gi techniques's by 80%, as well as Rabarta's by 90%.

## Technique Boosts

| Technique | Boost |
| ----------- | ----------- |
| Foie | Power +100% |
| Barta | Power +100% |
| Zonde | Power +100% |
| Gifoie | Power +80% |
| Gibarta | Power +80% |
| Gizonde | Power +80% |
| Rabarta | Power +90% |
