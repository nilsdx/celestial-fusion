# Paragon Frame

Paragon Frame is a rare Frame that boosts Shifta and Deband's level by 5, making level 35 reachable.

Paired with a Resta Power Boost, it is the best support Force armor in the game.

## Technique Boosts

| Technique | Boost |
| ----------- | ----------- |
| Resta | Power +800 |

## Set Effect

| Item | Effect |
| ----------- | ----------- |
| [Soul Devourer](/weapons/soul-devourer) | Hits: 1-2-5 (Normal/Heavy), 1-1-3 (Special), ATP +100, Animation: Custom, [Satellite Lizards](/enemies/satellite-lizard) and [Yowies](/enemies/yowie) can now be hit by this weapon |
