# Excalibur

Excalibur is a rare Saber. It is not a Destiny exclusive weapon, but it receives buffs when paired with [Tellusis](https://wiki.pioneer2.net/w/Tellusis).

## Set Effect

| Item | Effect |
| ----------- | ----------- |
| [Tellusis](https://wiki.pioneer2.net/w/Tellusis) | Hits: 2-1-3, Animation: Double Saber, Buffed Range & Angle (unknown values) |

## Availability

### Unsealing

Excalibur can be acquired by unsealing [Lame d'Argent](https://wiki.pioneer2.net/w/Lame_d%27Argent) by killing 10,000 enemies.
