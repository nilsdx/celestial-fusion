# Prophets of Motav

In combination with [Forbidden Grimoire](/units/forbidden-grimoire), this weapon can potentially cast Rabarta faster than a FOnewm/FOnewearl with [V802](/units/v802), and freezing becomes even more consistent when using [V803](/units/v803) on top.

## Technique Boosts

| Technique | Boost |
| ----------- | ----------- |
| Rafoie | Power +20% |
| Rabarta | Power +20% |
| Razonde | Power +20% |

## Set Effect

| Item | Effect |
| ----------- | ----------- |
| [Forbidden Grimoire](/units/forbidden-grimoire) | Expanded Range, Hits: 3-3-3 |

## Trivia
- Originally, this weapon's special was Jellen/Zalure Lv.5, but was changed to Rabarta to make it work with [Forbidden Grimoire](/units/forbidden-grimoire).
