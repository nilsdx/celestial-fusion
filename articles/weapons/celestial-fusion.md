# Celestial Fusion

Celestial Fusion are rare Mechguns. They boost both Jellen and Zalure range.

As they're equippable by all classes, they're considered to be the best Demon's weapon for both Hunters and Forces.

Their set effect with [Jointparts](/units/jointparts) gives tremendous range and accuracy boosts, as well as enabling Jellen+Zalure dual casting on Forces (whenever you cast Jellen or Zalure, both Jellen and Zalure will be casted at the same time).

## Technique Boost

| Technique | Boost |
| ----------- | ----------- |
| Jellen | Range +100% |
| Zalure | Range +100% |

## Set Effect

| Item | Effect |
| ----------- | ----------- |
| [Jointparts](/units/jointparts) | Range +90.0, ATA +20, MST+100, Dual Casting: Jellen + Zalure (FO only) |
