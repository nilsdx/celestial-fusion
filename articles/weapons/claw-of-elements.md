# Claw of Elements

Claw of Elements are rare Fists that boosts all Gi techniques by various amounts.

They're the best way to boost Gizonde.

## Technique Boost

| Technique | Boost |
| ----------- | ----------- |
| Gifoie | Power +60% |
| Gibarta | Power +90% |
| Gizonde | Power +90% |
