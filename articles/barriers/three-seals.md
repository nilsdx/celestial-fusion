## Variants

Three Seals has a cosmetic variant that can be acquired by combining it with another item to make Profound Darkness.

| Material | Result |
| ----------- | ----------- |
| [Orb of Illusion](/items/orb-of-illusion) | Profound Darkness |

## Trivia

- Profound Darkness's name is a reference to the true antagonist force from the Phantasy Star franchise. Its model is an edited version of [From the Depths](https://wiki.pioneer2.net/w/From_the_Depths), but depicting [Dark Falz](/enemies/dark-falz)'s second phase instead.
