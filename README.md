## About the project

## Stack

- Language: TypeScript
- Frameworks: React, Next.js
- Styling: TailwindCSS, and CSS3 for Markdown styling
- Icon libraries: Lucide React, React-Icons

## Run locally

### Prerequisites

- Node.js version 23 or later

### Installation

After cloning this repository, install all dependencies
```bash
npm i
```

Run the local server
```bash
npm run dev
```

Then access [127.0.0.1:3000](127.0.0.1:3000) to see the result.
